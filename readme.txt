Ejemplo de como dibujar con canvas y guardar en un directorio

Detalle:
Tenemos un cuadro donde hacer dibujo con el mouse o con el dedo en pantallas touch (u otro dispositivo como lapiz). Luego al guardar este se genera como archivo png en el directorio dibujos.

Utilizado:

-signature_pad
-JQuery
-PHP
